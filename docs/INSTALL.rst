wcc.facebook Installation
-------------------------

To install wcc.facebook using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.facebook`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.facebook

* Re-run buildout, e.g. with:

    $ ./bin/buildout
